NEURAL AI Tools Directory

What we're building

A curated directory of AI tools where users can browse, search, filter by category, and save their favourites. All data is stored in the browser using localStorage — no backend or database required. Built with plain HTML, CSS, and JavaScript as part of the ATU GUI & Web Development module.

01
Pre-loaded data
A JavaScript array of 12+ real AI tools lives in tools-data.js. Every page reads from this single source.
02
Dynamic rendering
JavaScript loops through the tools array and builds HTML cards on the fly. Filters and search update the DOM live without reloading the page.
03
Saved state
When a user saves a tool, its ID is stored in localStorage. The saved page reads those IDs and renders the matching tools.
HTML5 CSS3 — custom properties, grid, flexbox Vanilla JavaScript localStorage API URL parameters
Pages
Home index.html

Hero section with headline and search bar. Below it, a featured tools grid rendered by JavaScript. Each card has a Save button that stores the tool ID in localStorage and updates the navbar counter.

DOM — render tool cards localStorage — save on click
Browse browse.html

All tools in a filterable grid. Category filter buttons narrow the list live. A search input filters by name and description on every keypress. Both written by JavaScript updating the DOM without a page reload.

DOM — live filter & search localStorage — read saved state
Tool detail tool.html

Shows the full details of a single tool. Reads the tool ID from the URL parameter (tool.html?id=3), finds the matching object in the tools array, and populates the page elements with that data.

DOM — populate from URL param localStorage — save from detail page
Saved tools saved.html

Displays everything the user has bookmarked. Reads the saved IDs array from localStorage, looks up each tool, and renders the cards. Remove button on each card. Shows a friendly empty state when nothing is saved.

localStorage — read & remove DOM — render & update list
